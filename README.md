Mini-projet de jeu de dames en React avec Tailwind.

Run:

1. Installer les dépendances:

```bash
cd /path/to/checkers
npm install
```

2. Lancer en dev:

```bash
npm run dev
```

Fonctionnalités:
- Plateau 8x8, pions rouges/noirs, promotion en dame
- Mouvements, captures multiples obligatoires
- IA: easy/medium/hard/expert (random / heuristic / minimax / alphabeta)
- Historique et annulation
